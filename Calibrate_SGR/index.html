Project Summary:

Data Calibration:

As a part of the broader project lead by Luca Matone for analyzing LIGO data for SGR1806-20 Flare event that occurred on December 27, 2004 during LIGO Hanford 4km detector's astrowatch run, part of my task was to verify the data calibration at the relevant time and site. For details on this section, please click here.


Internal Code Review:

Columbia graduate student Peter Kalmus developed a search algorithm to search for gravitational waves associated with SGR events, and to set upper limits on the amplitude of potentially associated gravitational waves. The implemented analysis pipeline can search for gravitational wave signals associated SGR flares using either a single GWdetector data, or coherently two GWdetector data. This pipelinewill be used to analyze LIGO data collected during the SGR 1806-20 flare event other flare events that take place while one or more GW detectors are running for estimating upper limits on the amplitude of potentially associated gravitational waves. I closely worked with Peter as the internal reviewer of the code. The internal review document is currently available to GECo members only.


QPO Search:

The analysis pipeline developed by Luca Matone have used to analyze LIGO data during the SGR 1806-20 event, and it has been reviewed by the LSC. Currently I am working with Luca to enhance and extend the pipeline so that it can be used to analyze dual detector LIGO data to look for co-incident signature of Quasi Periodic Oscillations (QPOs) during GRB events originating from Soft Gamma Repeaters (SGRs). Further details on this project can be found here.
